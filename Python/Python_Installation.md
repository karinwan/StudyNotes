# Python

## Installation
1. Download Python from official [website](https://www.python.org/); 
2. (Windows) Modify system variable to add the PATH;
3. Run the command on terminal to check if Python is installed properly:
      ```
      python --version
      ```
      , or
      ```
      python3 --version
      ```
      , or for windows
      ```
      py --version
      ```
